Inbox is a free communication tool for WooCommerce vendors and customers. Vnedors can manage their own communication threads privately with this plugin. Any user who is logged in can ask a question or put any query through help section. User will select the department from list to which he has some query and provide his email which is optional and his message. These messages will appear in your inbox and admin panel as well. Relevant Administrators or Moderators will reply the queries as support departments. Admin can add new departments, manage staff department-wise. Admin can also monitor communication threads and apply filters with an alert.
WooCommerce customers also can manage their communication threads with support departments and online vendors. Only logged-in user can communicate within the system. This plugin is an internal communication system among website registered users. In addition you can create support/help desks like contact forms etc. unregistered users can also submit inquiries and your administrator/moderators can reply those threads.

= Tags =
messages, users, administration, support, tickets, resolution center, disputes, alerts, chat, help desk, vendors

###Features

* Admin can send/schedule messages to all users
* View all communication threads and apply filters
* Define different departments and assign staff for sales and support
* Shortcodes for users list
* Allow/Disallow words in threads
* Google Maps API compatibility
* Shortcode based pages
 
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There will be a link under WP Admin > Settings.


== Frequently Asked Questions ==

= Is this compatible with all WordPress themes? =

Yes, it is compatible with all WordPress themes which are developed according to the WordPress theme development standards.

= How can we use it? =
I.	Install the Plugin and Activate it
II.	Go to Admin Panel > Settings > Inbox
III. There are two readymade pages available in Pages & Short codes tab which can be inserted in relevant areas.
IV.	You can also use short code available in settings page, as per your requirements.
V.	In Communication tab, admin can view all conversations.
VI.	In Support Department tab you can add departments and manage staff as per departments.
VII. In Google API tab you can add your Location API, which will help customers to find you easily through Google Maps API. Latitude and Longitude will be calculated from your location string.
VIII. In Install Plugin tab you can install plugins which are required for this plugin. Mandatory and Optional are included.
IX.	In Advance Settings tab there are some advance short codes available.

= Is any other plugin required to use it? =

WordPress Bootstrap CSS plugin required for this plugin mainly.

= Can we communicate to multiple platform with this? =

It Can be used only for the same platform users.

= How can we select the user we want to send message? =

If You are a seller then on your product & profile/author page a contact button will be appeared. If you belong to support, admin or HR department then people will approach you through contact or help page. This plugin can be used for communication between seller and user + admin and user. 

= Does email address required to send messages? =

No, Email is optional for support page but account registration is required for the communication between seller and customer.

= Can we use it on any page or only on a specified page? =

Yes, you can use it on any page by just adding the short code to any page.

= Does it support instant chat feature? =

No Instant chat is not available in this version.

= Can non-registered user send any query or message by using this? =

Users can't send messages to each other without login, but a visitor can send message to support or admin from help page.

= Are all features being free in this plugin? =

Yes, all basic/required features are free in this plugin.

= What is the function of Google+ API? =

If you are a seller so you can add your location by Google Maps API. Customers can find you by distance preferences through your latitude and longitude values.
